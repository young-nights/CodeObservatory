# CodeObservatory Design Context

## Target Audience
Developers and engineering teams who use AI coding agents (Claude, Copilot, Cursor, Codex) to modify their codebase. They're technical, detail-oriented, and need visibility into black-box agent operations.

## Primary Use Cases
1. **Track agent changes**: See what files AI agents modified, when, and how
2. **File relationship graph**: Visualize project structure as an interactive constellation
3. **Version tracing**: Understand file evolution over time with timeline
4. **Impact analysis**: Quickly identify which files are connected to recent changes

## Design Direction: Precision Instrument
The interface should feel like a precision scientific instrument — restrained, precise, trustworthy. Beauty comes from precision, not decoration. It should evoke the calm focus of a well-calibrated instrument panel, using depth layering of color (not glass/blur) to create visual hierarchy.

## Brand Personality
- **Precision** — Like a scientific instrument (microscope, telescope, spectrometer)
- **Restraint** — Confidence to say more with less; no decorative excess
- **Trustworthy** — Clarity born from deliberate choices, not trend-chasing
- **Calm focus** — Reduces cognitive load, surfaces what matters

## Tone
- Editorially refined — like a premium scientific journal
- Confident restraint — nothing extraneous
- No emoji, no exclamation marks, no "fun" microcopy

## Differentiation
- NOT another generic dark-mode dashboard with neon accents
- NOT following glassmorphism/backdrop-blur trends
- Feels like a precision instrument: analytical, purposeful, quietly authoritative
- Drawing from: Carl Zeiss microscopes, Braun industrial design, IBM Selectric

## Design System References
- **Fonts**: Georgia (headings/serif) + system-ui (body) + SF Mono (code)
- **Color space**: OKLCH exclusively — no hex values
- **Palette**: Micro-blue tint (260° hue) — cool but warm-tint whites, never pure black
- **Spacing**: 4pt base grid, asymmetric hierarchy
- **Motion**: Exponential easing (quart-out/in), no bounce, no spring
- **Anti-patterns**: No backdrop-blur, no neon glows, no gradients, no particle effects
